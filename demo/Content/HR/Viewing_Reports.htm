    Viewing Reports

    The Amrita HMIS system provides several reporting features. As a HR personnel, you can generate multiple reports from the Report menu.

    This section provides information about the various reports you can generate with the Amrita HMIS application.

    1 Generating a report

    1. Click on the Report menu and select the report you want to generate.

    2. Click on the desired sub option in the report menu.

    The Amrita HIS Reports window appears.. It displays a set of criteria or parameters you can select from to generate the report.

    3. Select the criteria you want to define and enter the corresponding values.

    4. Specify the type of report (PDF/CSV) you want to generate.

    5. Click Run Report to produce a report

    The AHMIS application processes your request and eventually displays the requested report.

    2 Department Wise Employee List

    Select this report to view the list of employees department wise. Specify the input parameters HR_Departments and Designation.

    3 Department Wise Employee List

    Select this report to view the job history of a particular employee. Specify the input parameters EMP ID (i.e. employee ID), HR_Departments. The report provides Emp ID, Employee Name, Start Date, End Date, Department and Designation.

    4 Daily Absent Report

    Select this report to view the absentees on a particular date. Specify the input parameters Date and HR_Departments. The report provides Emp ID, Employee Name, Department and Designation.

    5 Department Wise Regular Absentees

    Select this report to view the regular absentees, department wise absentees on a particular date. Specify the input parameters From Date, To Date and HR_Departments.

    6 Pending Increment Report

    Select this report to view the employees whose increment is pending. Specify the input parameters From Date, To Date and HR_Departments. The report provides the Emp ID, Department, Employee Name, Designation and Increment Date.

    7 Pending Appraisal Report

    Select this report to view the employees whose appraisal is pending. Specify the input parameters From Date, To Date and HR_Departments. The report provides the Emp ID, Department, Employee Name, Designation and Appraisal Date.

    8 Department Wise Consolidated Salary

    Select this report to view the department wise consolidate salary for a month. Specify the input parameters HR_Departments, Month, Year. The report provides the Designation, Department and Total Salary.

    9 Employee PF Yearly Contribution

    Select this report to view the yearly PF contribution of employees and employer. Specify the input parameters From Date, To Date and Location.

    10 Employee PF Monthly Wise

    Select this report to view the monthly PF contribution of a particular employee. Specify the input parameters Emp ID, From Date and To Date.

    11 Employee Leaving Service Details

    Select this report to view the list of employees leaving service during a date range. Specify the input parameters From Date, To Date and Location. The report provides Account Number, Name of Member, Name of Parent/Spouse, Date of Leaving and Reason for Leaving Service.

    12 Employee PF Joined Details

    Select this report to view the details of employee that have PF. Specify the input parameters From Date, To Date and Location.

    13 HR Payslip

    Select this report to view payslip details for specific pay run IDs. Specify the input parameter PayrunID.
