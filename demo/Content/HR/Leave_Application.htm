    Submitting Leave Application

    Leave Application Window

    FieldsDescription
    Leave Request NumberDisplays the leave request number that is automatically generated once a leave request is submitted.
    Leave Request DateEnter the date on which the leave is requested.
    Employee NumberEnter the employee number.
    NameDisplays the name of the employee.
    GenderDisplays the gender of the employee.
    Leave StatusDisplays the status of the leave post submission.
    DepartmentDisplays the department of the employee.
    DesignationDisplays the employee’s position in the hospital.
    ApplicationClick this tab to enter the leave application details.
    Leave TypeLeave types may be casual, LOP etc.
    Reason For LeaveEnter the reason for taking leave.
    Start DateEnter the date on which the leave begins.
    End DateEnter the date on which the leave ends.
    Number Of LeavesEnter the total days of leave.
    ApprovalClick this tab to enter the leave approval details.
    CancellationClick this tab to enter the leave cancellation details.
    EligibilityClick this tab to view the eligibility of the employee for leave.
    Past LeavesClick this tab to view the past leaves of the employee.
    Medical FitnessClick this tab to enter the medical fitness details of the employee.
    Add NewClick this to save the leave application.
    UpdateClick this to update the leave application.
    ClearClick this to clean out the screen.

     

    To Submit Leave:

    1. Select Leave Application from the sub menu, Leave of HR main menu.

    Leave Application screen appears.

    2. Enter the employee number.

    The Leave request date, name, gender, leave status, department and designation are automatically displayed by the AHMIS application.

    3. Click the tab – Application.

     Enter the following details:

     Leave Type

     Reason for leave

     Start Date

     End Date

    4. Click Add New to save the leave application.

    A confirmation message is displayed at the top of the screen.

    NOTE:After submitting the leave application the status changes to pending.

    Approving Leave

    1. Click the Approval tab in the Leave Application screen.

    2. Enter the following details:

     Supervisor comments

     HRD Comments

     HOD comments

    3. Click Approve.

    A confirmation message is displayed at the top of the screen and the status changes to Approved.

    Rejecting Leave

    1. Click the Approval tab in the Leave Application screen.

    2. Enter the following details:

     Supervisor comments

     HRD Comments

     HOD comments

    3. Click Reject.

    A confirmation message is displayed at the top of the screen and the status changes to Rejected.

    Cancelling Leave

    1. Click the Cancellation tab in the Leave Application screen.

    2. Enter the following details:

     Reason for cancellation

     Cancellation Date

    3. Click Cancel Leave.

    A confirmation message is displayed at the top of the screen and the status changes to Cancelled

    Viewing Leave Eligibility

    • Click the Eligibility tab in the Leave Application screen.

    Viewing Past Leave of the Employee

    • Click the Past Leaves tab in the Leave Application screen.

    Entering Medical Fitness Details of the Employee

    1. Click the Medical Fitness tab in the Leave Application screen.

    2. Enter the following details:

     Issue No

     Issued authority

     Fit to join on

     Issued Date

     Issued authority details

     Medical details

    3. Click Save.

    A confirmation message is displayed at the top of the screen.
