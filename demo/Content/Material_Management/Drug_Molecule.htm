    Drug Molecule Entry

    The following fields are available in the screen

    Drug Molecule Name (text box)

    Description (text box)

    Drug Base class (drop down)

    Drug Sub class (drop down)

    Drug Category (drop down)

    Active (check box)

    First User has to enter the exact molecule name in the Drug Molecule text box. While adding new molecule User has to follow some naming conventions. That is all the characters should be entered in capital letters or first character in capital and other remaining characters should be in small. Whenever new name is added the same naming convention should be followed. If it is entered like this means at the time of displaying the names in alphabetical order or for searching molecules it will be more useful and system fetch you exact search results.

    After entering the molecule name User has to press the TAB key in the keyboard. Immediately control will moves to Description text box and by default detailed entered in the name field will be displayed in the description also. User can change the description detail if needed.

    Each molecule is categorized based on three things. They are Base Class, Sub Class and Category Class. In the Base class molecules are categorized based on different type of treatments or different type of specialities. It means a molecule may be used for Cardiac related problems or Nervous System or Ophthalmic related problems etc. Next level of categorization is called Sub class. This classification is nothing but which parts of body the molecule is affecting or what are all the various sub sections of the selected Section. The third level of categorization is purpose of the molecule or the molecule heels which type of problems etc. Again the above information is just an example only. User can create any type of Base class, Sub class and Category class up to their convenient. Only thing is Category is depends on Sub class and Sub class is depends on Base class. These 3 levels of categorization is provided to help the User in searching the medicine based on Generic name if required brand medicine is not available.

    Separate master screens are provided to capture details of Base class, Sub class and Category class.

    Now after entering the required description User has to select one of the option from Drug Base Class drop down. User must know complete details about the new molecule. As soon as one of the option is selected in the Base class, Drug Sub class drop down is automatically refreshed and it will display options belongs to selected base class. Then User has to select the appropriate Sub class. As soon as sub class is selected Drug Category class drop down is automatically refreshed and User has to select the appropriate Category class.

    Finally Active check box should be selected indicating that we are going to Use this molecule. If the check box is not selected means the newly added molecule can’t be used anywhere. Since it is very important field separate privilege is provided for this field. Only privileged Users can able to update this field.

    After filling all the details “Add New” button should be pressed. On successful completion of this process system will give confirmation message. If any of the details are not filled except Active check box system won’t allow the User to save the details. It will display error message. It means all the fields in the screen are mandatory.

    Suppose if User wants to put more than one molecule name together then to concatenate the molecules + (plus) symbol should be used. Consider the following example.

    Assume that User wants to enter the molecules Sabutamol and Bromhexine together as single molecule means it should be entered like this.

    Sabutamol+Bromhexine

    Suppose if User wants to enter 3 molecules together means it should be entered like this.

    Sabutamol+Bromhexine+Guaiphensin

    While entering more than one molecule together User should not give the space before or after the + (plus) symbol. Name convention should be strictly followed for better search results. Presentation and strength details should not be added with molecule name.

    Any point of time details of a molecule can be changed. In order to do so first User has to press the Molecule Browser button. Then another small window will be opened through which User can search the require molecule name by just entering few characters and pressing the search button. System will display all the molecule names which satisfies the given condition. From that User can select the required molecule name by pressing the “Select” button. Immediately the selected molecule will be displayed in the first screen and the newly opened window will be automatically closed. Now User can change the name, description, base class, sub class and category class whichever is needed. Finally by pressing the “Modify” button newly updated details will be saved successfully.
