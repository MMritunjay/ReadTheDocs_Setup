    DRUG BASE CLASS

    This is basis of all the drug categorization. Name of the drug refers to the name of the functional area of the drug i.e. the core area the drug affects.

    FIELDS

    1) Drug Base Class Name (Text Box)

    2) Description (Text Box)

    3) Active (Check Box)

    ACTION BUTTONS

    1) Create

    2) Search

    3) Update visible only after displaying the values by search button)

    4) Close

    5) Clear

    USER OPERATIONS:

    1) CREATING A NEW ENTRY IN DRUG BASE CLASS

    2) SEARCHING FOR A PARTICULAR ENTRY IN DRUG BASE CLASS

    3) UPDATING THE CONTENTS IF NECESSARY

    4) CLEARING THE ENTIRE CONTENTS WHICH ARE DISPLAYED ON THE SCREEN

    5) CLOSING THE SCREEN

    1)CREATING A NEW ENTRY IN DRUG BASE CLASS

    NoStepsExample
      Suppose the core area or the main functionality of the drug is to alleviate pain and fever’
    1Open 'Drug Masters' from the Pharmacy Masters menu (masters login) 
    2Enter values in the drug base class name and description fieldsThe user can enter drugbaseclass name as Pain Fever, Give the description as Pain Fever
    3check whether the newly entered details is active or notCheck the active check box If the item is active
    4Click on the create button 

    The message

    “Created a base class entry with id (*) “

    will be displayed

    The can’t do’s in the operations

    We cant create a drugbaseclass with a name already existing

    2)SEARCHING FOR A PARTICULAR ENTRY IN DRUG BASE CLASS

    Searching is a very simple process which can be done with much easiness

    We can search for an item by entering the required values and by clicking the search button

    The display will be based o n the criteria Blank search (searching without entering any criteria)will display the inactive items only

    3) UDATING THE CONTENTS IF NECESSARY

    After listing the elements there may a need for changing the name

    description or making the item active or inactive etc. This is done using update button

    Can be done using

    Select the line item by clicking the edit button

    The selected item will be displayed on the above window.

    Make the desirable changes

    Click on the update button

    3)CLEARING THE ENTIRE CONTENTS WHICH ARE DISPLAYED ON THE SCREEN

    Clearing the entire contents displayed can be done by pressing the clear button

    4) CLOSING THE SCREEN

    This is for closing the screen by simply pressing the close button
