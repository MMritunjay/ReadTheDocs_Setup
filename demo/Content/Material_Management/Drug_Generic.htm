    Adding Generic Drug Details

    After creating a molecule, we have to create Generic names of that molecule. Generic name is nothing but combination of Molecule Name, Strength and Presentation. User could either create Generic Names from Drug Molecule tab in 'Drug Masters' screen itself or from the 'Drug Generic' tab in the Drug Masters screen.

    If Generic is creating from Drug Molecule screen, Click the “Assign Drug Generic” button, after selecting the required molecule through Molecule browser screen. After pressing the “Assign Drug Generic” button, system will open a new window for adding new generic name. The fields available in the screen are

    Molecule description (text box)

    Generic code (text box)

    Drug Strength (text box)

    Drug Strength Unit (drop down)

    Drug Presentation (drop down)

    Active (check box)

    By default Molecule description will be automatically displayed. Then User has to enter some code for that Generic name in the “Generic Code”. User can enter alphanumeric characters as code value but special characters are not allowed.

    Then in the Drug Strength column User has to enter the exact strength of that medicine in numbers. Since it is a text box it can accept characters also. It is up to User what value is to be entered in that column.

    Next User has to select the strength unit from the corresponding drop down.

    Then Use has to select the appropriate presentation from the drop down. By default Active check box is always selected.

    Finally by pressing the “Save” button at the bottom of the screen the newly added generic name will be displayed in the table immediately.

    Consider the following examples in entering new generic names.

    Creating generic name for single molecule. Assume that molecule name is Sabutamol, strength is 50, unit is mg and presentation is tablet. Assume that generic code will be TSAB50.

    Generic code : TSAB50

    Drug Strength : 50

    Drug Unit : mg

    Presentation : Tablet

    After adding the above details the generic name is created like this.

    Sabutamol (50 mg) Tablet

    Creating generic name for two molecules with same strength. Assume that molecule names are Sabutamol and Bromhexine. Its strength is 100 mg.

    Generic code : TSB100

    Drug Strength : 100+100

    Drug unit : mg

    Presentation : Tablet

    After adding the above details the generic name is created like this.

    Sabutamol+Bromhexine (100+100 mg)Tablet

    Look into the Drug strength column how it should be entered. Since there are two molecules are involved so strength should be mentioned for each molecule separately.

    Creating generic name for 3 molecules of different strengths. Assume that molecules are Sabutamol, Bromhexine and Guaiphenesin and their strengths are 50, 75, 100 respectively and their unit is mg.

    Drug Code : TSBG1

    Drug Strength : 50+75+100

    Drug unit : mg

    Presentation : Tablet

    After adding the above details the generic name is created like this.

    Sabutamol+Bromhexine+Guaiphenesin (50+75+100 mg)Tablet

    Like the above details User can create any number of combinations of generic name for same molecule. It means User can create generic name of same molecule with different combinations of strength, unit and presentation.

    Sabutamol (50 mg)Tablet

    Sabutamol (50 mg)Capsule

    Sabutamol (100 mg)Injection

    This generic name so important at the time of creating new Drug Item (Brand name). At that time proper generic name should be selected from the available drop down.

    After saving the details newly created generic name is displayed in the table along with “Edit” button option. By pressing the “Edit” button the selected generic name details are displayed in the respective columns at the top of the screen. User can change any of the details at any time and by pressing the “Modify Generic” button altered details are captured and immediately User can see the changed details in the table.

    Our thought process for adding the non-formulary was:

    1) A hospital may decide certain brands as not in the formulary

    2) Also, a hospital may decide certain generic not in the formulary and that is the reason we added a flag in the drug_generic table.
