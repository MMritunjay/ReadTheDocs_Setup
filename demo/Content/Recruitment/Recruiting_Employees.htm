    Recruiting Employees

    The following sections introduce the concepts and provide instructions on using the AHMIS application's HR module for recruitment process.

    The Recruitment Work flow:

    You must first record the details of the vacancy and get the new manpower request approved before advertising for prospective candidates.

    Following figure shows the recruitment work flow. You can perform all the operations presented in the work flow using the AHMIS application.

    Key Features

    Following are the major tasks you can perform with the AHMIS registration module

    • Create new manpower requests at the department level

    • Approve of selected manpower requests

    • Merge multiple requests in case they are requesting to fill vacancies for the same position.

    • Export selected requests' details from AHIS

    • Assign one or more recruitments to a manpower request

    • Create and map candidate evaluation questionnaire and scoring rules to a re-cruitment

    • Import applicant (candidate CVs) data into AHIS

    • View selected candidate CVs

    • Evaluate candidate by selecting the appropriate marks in the questionnaire

    • View the final scores of the candidates (calculated by the application as per user settings)

    • Tag the selected candidates and record the marks and interview details

    • Generate appointment letters for the selected candidates

    • Generate reports

    Logging In

    To use the AHMIS system, you must first login to the module. To log in, you must use the username and password provided to you.

    To login to the Amrita HMIS system:

    Click on the Amrita HMIS Software icon on your desktop to display the login page.

    1 Enter the user name and password.

    2 Click Enter to log on.

    3 Click OK

    The AHMIS Home page appears as shown in Figure 3 below.

    Click on the Registration menu to start the registration process.

    To reopen the pane click in the top left corner of the screen above

    In Alerts and Notifications, a selected notification can be forwarded to other uses by clicking on for the notification message.

    In the resulting pop up window, specify the specialty/doctor and click Forward.

    Home page setting

    User can specify the Home page he wished to see first every time he logs in.

    1. Click open the desired page

    2. In the top row of menus on the screen (beside Log out button) click on the down arrow beside Home icon.

     

    1. From the 2 given options, click Use Current as my home page option.

    There after, every time the particular user logs in, the newly selected Home page will be displayed.

    Clicking on icon will open the Location Selector page, where the user can change the current location – service center/service point/store/nursing station etc.
