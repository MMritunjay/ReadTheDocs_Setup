    Manpower Requests

    Manpower requests are raised by the privileged users – generally the Heads Of De-partments of various clinical and administrative departments in the hospital. A request is a form where requester has to fill in the details of the position that is currently vacant and is requiring to be filled by one or more new recruits. The list of criteria to be satisfied by the recruits are given – for e.g. Qualification level, work experience, any gender preferences, preferred age-range etc. Also given are the details of the position – the salary, shift timings etc and finally by when the resources are required.

    The requests created by all department HODs are received by the Human Resource department Manager. The HR Manager has the privilege of approving or sanctioning few or all of the resources requested.

    Creating a Manpower Request

    You can create a manpower request if you are the head off the department and have been assigned the privilege to create requests. You can create a new request using the following steps.

    1. From HR Recruitment menu select Manpower Request Posting sub menu.

    Here, select New Manpower Request

    1 New Manpower Request page opens.

    1. Select the details of the request such as Position, No: of vacancies, Required by date, salary, qualification and experience, gender preference (if any ) etc

    Manpower requests saved by all department users are sent directly to the HR Manager who will have the privilege to edit and approve or reject these requests.

    1 From HR Recruitment menu select Manpower Request Posting sub menu.

    Here, select Manpower Request Browser

    2 Manpower Request Browser page opens listing all requests received by the user so far

    3 Enter search criteria - such as Department, Position, work area etc

    4 Click Search

    5 Requests satisfying the given criteria are listed out

    6 Click on the icon in column “Action” for the desired request

    7 Edit Manpower Request page opens

    8 Enter the no: of vacancies approved in the (mandatory) field – Approved Vacancies

    9 Make any necessary changes

    10 Select the approval status from the drop down list

    11 Click Update

    12 Please note that the approval status can also be set to “Reject” thereby rejecting the request.

    You may also save changes in the request now and then select an approval status at a later time.

    The changes made in a request are available for tracking purposes to the user who raised the request (in My Requests)

    1.1.1 Bulk Approval of Requests

    To approve multiple requests at one go, do the following:

    1 From HR Recruitment menu select Manpower Request Posting sub menu.

    Here, select Manpower Request Browser

    2 Manpower Request Browser page opens listing all requests received by the user so far

    3 Enter search criteria - such as Department, Position, work area etc

    4 Click Search

    5 Requests satisfying the given criteria are listed out

    6 Click in the select check boxes beside the requests that are to be approved

    7 Click Approve

    8

    9 In the case of bulk approval, the no: of requested vacancies is accepted automati-cally in the no: of approved vacancies field (unless a value was already entered in the approved vacancies field)

    (Please note that requests can also be rejected through the same process by clicking the Reject button in step 6 above)
