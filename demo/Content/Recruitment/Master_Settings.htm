    Master Settings

    In Amrita Recruitment module, the following details can be added to help evaluate candidates for all positions at the hospital at any point of time. Questionnaires can be prepared with multiple choice and/or descriptive questions. In case of multiple choice questions, marks to be awarded to each choice of a question can be set

    These settings may be considered as “one-time” meaning that once typed in and saved, these settings can be adopted for all future recruitments unless you wish to customize further.

    This feature affords the user maximum flexibility in building evaluation systems.

    Manage Evaluation Scale

    Here, the user can create a new evaluation scale as well as view all the scales created in this institution till now. Edit option is also provided to modify any existing scale’s name.

    1 From menu HR Recruitment select sub menu Settings and further, Evaluation Settings

    Evaluation Scale page tab opens (by default), listing out the scale names already captured. (The other 2 tabs present here are Evaluation Scale Options and Evaluation Questions)

    1 Enter the name of the the new scale

    2 Click Save

    Confirmation message appears stating successful capture of details

    Scale Options

    After naming the scale, you move on to Evaluation Scale Options where the stan-dard scores can be listed. For e.g. if the reviewer selects the option “Good” as per the Likert scale shown in the image below, for a particular answer , then the answer has scored 3 marks. “Poor” is assigned 1 mark and ‘Excellent” assigned 5 marks.

    In case you wants to score in number format, a new scale can be defined where option “10” can be assigned Scale option index “10”.

    1 From menu HR Recruitment select sub menu Settings and further, Evaluation Settings

    Evaluation Scale page tab opens (by default),

    1 Click on tab Evaluation Scale Options

    Evaluation Scale Options page opens, listing out the saved scale names and the assigned options

    1 Select the the scale for which options are to be added

    2 Enter the text to be displayed as option (“Scale option”) and the score to be used for that option (“Scale option index”)

    3 Click Save

    Confirmation message appears stating successful capture of details

    Add Questions

    Once the scoring pattern is decided, move on Evaluation Question, to view the criteria or questions that are already saved for the organization in the application, with associated scoring pattern. The user may add any further question to this list for judging the proposals submitted for a particular notification.

    1 From menu HR Recruitment select sub menu Settings and further, Evaluation Settings

    Evaluation Scale page tab opens (by default),

    1 Click tab Evaluation Questions

    Evaluation Questions page opens, listing out the saved questions and their assigned scoring rules.

    1 Select the Evaluation scale

    2 Enter the question

    3 Click Save

    Confirmation message appears stating successful capture of details
