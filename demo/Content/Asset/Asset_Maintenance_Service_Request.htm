    Asset Maintenance

    Fixed assets are to be maintained properly. Work orders have to be created for every asset that requires maintenance.

    Entering Asset Service Request

    To complaint about a problem with an asset, you need to create a complaint request.

    To create a complaint request:

    • From the Asset menu, select Service Request.

    ◦ A request number is automatically generated for the request and dis-played in the window.

    • In the Request Description field, describe the problem with the asset.

    • Enter the Asset Category.

    • Select the priority of the problem: high, medium, or low.

    • Enter contact information.

    • Select the Service Request Status: Open, Closed, Pending.

    • Enter the Servicing Department

    • Enter the date on which the request is made.

    Use Create Complaint Request to complete the process

    The complaint request is created and forwarded to the department that is responsible for fixing the problem. Please see the screen shot below.

    Modifying Service Requests

    To modify a service request:

    1. Click Service Request Browser in the Asset menu.

    2. Enter any of the search criteria and use Search

    2. Use Edit of the required.
