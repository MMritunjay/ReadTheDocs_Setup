    Asset Status

    Status indicates the current state of the asset. Assigning of various statuses to assets should be done carefully by respective privilege users, since various reports are generated by the management with the asset status as critical value. Currently the following statuses are there:-

    • CANCELLED.

    • CONDEMNED

    • DAMAGED

    • DISPOSED

    • FOR CONDEMNATION

    • INSTALLATION

    • MISSING

    • NOT WORKING

    • NOT IN USE

    • RETURNED-BB

    • RETURNED-EX

    • REPAIR OUTSIDE

    • IN REPAIR

    • RETURNED-RJ

    • RETURNED-WR

    • IN SERVICE

    • IN STORE

    • TRANSFERRED OUTSIDE

    Notes:-

    1. Values in the Department combo in the page is coming from the ‘De-partment Masters’ in HIS. In case new entries are needed, it has to be done through the said master page by a privileged user.

    2. Values in the Category and Sub-category combos are coming from the Category master. Please see Section 7.3 for details

    3. Values in Status combo is coming from the LookUp Browser; please see Section 7.1 for more details

    4. Values in the Location combo is coming from the Location master in HIS. In case new entries are needed, it has to be done through the said master page by a privileged user.

    5. Values in the Custodian Dept combo is coming from the ‘Department Masters’ in HIS. In case new entries are needed, it has to be done through the said master page by a privileged user.

    6. Values in the Custodian combo is coming from Staff Master. The names listed in the combo are of employees assigned with Staff Function ‘ASSET CUSTODIAN’. If more employees need to be added to the custodian list, it has to be done through the said master page by a privileged user.

    7. Shelf Life – Its a default data provided in the system, no master is availa-ble

    8. Values in Depreciation combo is coming from the LookUp Browser; please see Section 7.1 for more details

    Once the asset is saved in to the asset module in HIS, the ‘+’ mark that was earlier seen in the ‘Material Received Item Browser’ will get disappeared, as shown in the screen shot below.

    The system has the facility to define Asset Department, Asset Custodian department and Asset Custodian. Custodian can be assigned to Individual assets. Department to which asset belongs and its custodial department may be different. In this case even though the asset belongs to the GI Surgery, its custodian department is set as Anaesthesiology, since it is an anaesthesia machine and upkeep of the same is done by the Anaesthesiology department.
