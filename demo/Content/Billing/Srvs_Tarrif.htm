    Browsing for Patients and Bills

    The Amrita HMIS Billing services provide several features to help you quickly search for a patient or a bill.

     

    1 Browsing an IP Patient

    You can search for the patient admitted in the hospital using the AHMIS application.

    1. From the IP Patients menu, select IP Patient.

    The IP Patient Browser screen appears.

    2. Enter any of the following search criteria:

     MRD Number

     Name

     Specialty

     Admitting Doctor

     Status

     Nursing Station

     Ward No

    3. Click Search.

    The IP patient details such as name, MRD, ward, bed no, sex, age, patient category, consulting doctor, specialty appears on the screen.

    2 Browsing a Registered Patient

    You can search for a patient registered in the hospital, regardless of IP, OP, and ER using the Advanced Patient Search option of AHMIS application.

    1. From the IP Patients menu, select Advanced Patient Search.

    The Advanced Patient Search screen appears.

    2. Enter any of the following search criteria:

     MRD Number

     Full Name

     First Name

     Middle Name

     Last Name

     House Name

     Address2

     Post Office/ Town

     District

     State

     Country

     Registration Date From

     Registration Date To

     Date of Birth

     Sex

    3. Select option to Satisfy i.e. All or Any to the above set of criteria. By selecting All, you are forcing the search to satisfy all the criteria. Instead it is wise to select Any for most searches.

    4. Check the visit status Active to search for the active patients and check All to search for both active and inactive patients.

    5. Check the patient status IP to check for inpatients, OP for out patients, ER for emergency patients and All for the entire registered patients.

    6. Click Search.

    The patients meeting the search criteria appear on the screen.

    3 Browsing Service Tarrifs

    You can search for the charges of various services offered by the hospital. There are various billing types for a single service. For example the service charge for nursing home from general wards. There is a base price for each service and based on this various billing types for the service are defined. No charges are involved with prisoner’s wards and parsee wards.

    1. Select Service Tariffs from the Billing menu.

    The Service Browser screen appears.

    2. Enter any of the following search criteria:

     Service Code

     Service Name

     Category

     Subcategory

    3. Check Show Schedulable to search for the scheduled services.

    4. Check Show Packages to search for the packaged services.

    5. Click Search Services.

    The details of the services meeting the search criteria appear on the screen. The details include Service Name, CPT Code, Base Price, and Service Category.

    6. Click Show Prices to view the prices of the service based on different billing types.

    Billing Type Prices screen appears with different type and prices of the service.

    7. Click Close to close the Billing Type Prices screen.

    4 Browsing Registration Receipts

    At the time of registration, a registration fee is collected from the patient. The Registration Receipts window helps you generate a receipt for the amount paid by the patient. You can search for the registration payments made based on the patient’s MRD number, Receipt No, From Date, To Date, and so on.

    1. From the Billing menu, select Registration Receipts.

    The Registration Payment Receipts screen appears.

    2. Enter any of the following search criteria to search for the registration receipts:

     MRD Number

     Receipt Number

     From Date

     To Date

    3. Check For Current Visit Only to check for the registration receipts of present visit.

    4. Click Search to search for the receipt.

    The HMIS application searches for the receipt and displays the details in the window.

    5. Click Print to print a receipt.

    6. The registration receipt opens up in PDF format.

    5 Browsing Advance Payment Receipts

    IP patients in the nursing home pay an advance amount when they are admitted into the hospital. Their service charges are automatically debited from the advance amount they pay. The Advance Receipts feature allows you to generate a receipt for the advance payments made by the patients. You can search for the advance payments made by a patient using several criteria, including the MRD number, From Date, To Date, and Receipt number.

    1. From the Billing menu, select Advance Receipts.

    The Advance Payment Receipts screen appears.

    2. Enter any of the following search criteria to search for the advance receipts:

     MRD Number

     Receipt Number

     From Date

     To Date

    3. Check For Current Visit Only to check for the advance receipts of present visit.

    4. Click Search to search for the receipt.

    The HMIS application searches for the receipt and displays the details in the window.

    5. Click Print to print a receipt.

    6. The advance receipt opens up in PDF format.

    6 Browsing Bill Payment Receipts

    Use the Bill Payment Receipts feature to generate receipts for the various IP, OP, and material bills paid by the patient. You can search for specific Bill Payment Receipts using the patient’s MRD number, Receipt No, or those within specified periods of time.

    1. From the Billing menu, select Bill Payment Receipts.

    The Bill Payment Receipts screen appears.

    2. Enter any of the following search criteria to search for the bill payment receipts:

     MRD Number

     Receipt Number

     From Date

     To Date

    3. Check For Current Visit Only to check for the bill payment receipts of present visit.

    4. Click Search to search for the receipt.

    The HMIS application searches for the receipt and displays the details in the window.

    5. Click Print to print a receipt.

    The bill payment receipt opens up in PDF format.

    7 Browsing Refund Receipts

    Use the Refund Receipts feature to generate receipts for the amount refunded to the patient. You can search for specific Refund Receipts using the patient’s MRD number, Receipt No, or those within specified period of time.

    1. From the Billing menu, select Refund Receipts.

    The Refund Payment Receipts screen appears.

    2. Enter any of the following search criteria to search for the bill payment receipts:

     MRD Number

     Receipt Number

     From Date

     To Date

    3. Check For Current Visit Only to check for the refund receipts of present visit.

    4. Click Search to search for the receipt.

    The HMIS application searches for the receipt and displays the details in the window.

    5. Click Print to print a receipt.

    6. The refund receipt opens up in PDF format.

    8 Browsing Bill

    Use the Bill Browser feature to search for different types of bill created for the patient. You can search for any bill using the patient’s MRD number, visit code, bill type, bill status, bill payment status or those within specified periods of time.

    1. From the Billing menu, select Bill Browser.

    The Bill Browser screen appears.

    2. Enter any of the following search criteria:

     Bill Number

     MRD Number

     Visit Code

     From Date

     To Date

     Bill Type

     Bill Status

     Bill Payment Status

    3. Click Search to search for the bill.

    The HMIS application searches for the bill and displays the details in the window.
