    Entering Journal Voucher

    1. From the Transactions main menu select Voucher Entry.

    2. Select the Voucher Type as Journal from the drop down.

    Now a new panel appears.

    3. The voucher page displayed is in 3 partitions. The top one show fields Voucher Number, Date (Denotes Voucher Date), Invoice Number, Invoice Date, and Due Date (Denotes Invoice Due Date). The second partition shows Business Area, Profit Centre/ Cost Centre, Debit/ Credit, Amount, Narration and an Add button. The third partition shows fields Dr., Cr., Diff. and buttons Search, Clear and Close.

    4. Fill in the following details on the first partition:

    • Voucher Date

    • Invoice Number

    • Invoice Date

    • Invoice Due Date

    5. Select the Business Area, Profit Centre, Cost Centre, Debit or Credit from the respective dropdowns in the second partition.

    6. Enter the Amount and Narration as required.

    7. Click Add button.

    Now, two more buttons Delete and Cancel can be seen in the second partition. The data table in the second partition is updated with the entry. The Dr. or Cr. and Diff. fields get updated.

    8. Repeat Steps 5, 6 and 7 *

    9. Click Save button to add the voucher entry.

    System checks whether the Debit and Credit are equal; if not throws a message ‘Debit and Credit are not equal’, and the voucher is not saved. if yes, the voucher is saved. The added voucher entry is displayed in the third partition.

    10. A voucher number is generated and shown against the ‘Voucher Number’ field.

    11. A pop up message ‘Do you want to print this report’.

    12. Click ‘Cancel’ if you do not want to print it immediately.

    13. Click Ok to print, to get the printer interface.

    14. Click Ok, the voucher gets printed.

    15. Buttons Print Report and View Report can be seen along with buttons Search, Clear and Close.

    16. Print Report will direct you to the printer interface and view report will give you a print preview of the voucher.

    17. Click Clear, and user will be able to do a fresh contra voucher on the same voucher date. Click Close, the contra voucher format is closed and user can now select a new voucher type from the voucher type combo.

    * Narration needs to be entered only once. This will be copied to all other narration fields.
