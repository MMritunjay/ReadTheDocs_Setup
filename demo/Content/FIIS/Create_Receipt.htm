    Create Receipt

    The main receipts as far as hospitals are concerned are from the patients and the third party payers. Such kind of collections is recorded in the Patient Billing sub-system. But there are miscellaneous receipts also. Like wise advance refunds can also be settled using this. These can be recorded using the new ‘Create Receipt’ page. The steps to be followed are as follows.

    1. Select sub-menu ‘Create Receipt’ from the main menu ‘Transactions’

    2. A page titled ‘Create Receipt’ appears with fields ‘Advance Category’, Date, Instrument Date, Instrument Type, Instrument No., Cash/Bank A/c, Paid By, Amount, and Remarks. Balance fields on either side below Sales A/c and To A/c shows balances of the two accounts and are read only fields. Two buttons ‘Save’ and ‘Clear’ are also seen.

    3. Fill the relevant fields and click ‘Save’

    4. A message ‘Do you want to settle the receipt with any existing Bills?’ pops up

    5. Click OK to settle it with existing bills, Click Cancel to do it later.

    6. New Buttons Update, Print Report and View Report appears along side the button Clear

    7. Click ‘Print Report’ to print the voucher. Click ‘View Report’ to view the voucher. Click ‘Clear’ to clear the screen and create a new invoice

    8. Use ‘Update’ to modify the details already entered.

    9. If the receipt is for settling the advance taken earlier, select any one of the values from the Advance Category before doing the entry. As in the case specified for ‘Advance payments’, the system dynamically brings in the required fields to capture necessary data.

    10. Once payment is saved the system will prompt the user to settle the amount against the payment made earlier by automatically bringing the settlement page.
