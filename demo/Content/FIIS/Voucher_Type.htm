    Creating Voucher Type

    Details to be captured for different transactions may be different. For Ex:- for recording a purchase transaction fields like ‘Purchase Order Number’, ‘Purchase Order Date’, ‘Invoice Number’, ‘Invoice Number Date’, ‘Purchase Account etc:- where as a contra entry requires a different set of details to be captured. To cater this requirement, the Amrita FIIS provides five different predefined voucher formats or predefined types of vouchers. They are ‘Contra’, Journal, ‘Purchase’, ‘Payment’, and ‘Receipt. These are used for recording various transactions. The system allows creation of new voucher types under any of these voucher Types.

    To create a new voucher Type:

    1. From the Masters main menu select, Voucher Type.

    The Master page with two divisions appears. One on the Left Hand Side (LHS) is named Voucher Master and the other on the Right Hand Side (RHS) is named Voucher Type Master. Voucher Types already existing is listed on the LHS where as fields and buttons to add new voucher types and edit existing voucher types is available on the RHS.

    2. Click Add to add a new voucher type.

    Now the Add, Edit and Delete button disappears and Update and Cancel button appears.

    3. Select the primary voucher from the drop down Under.

    4. Enter the Name of the voucher.

    5. Check the Authorization check box if the voucher needs to be authorized.

    6. Enter the Prefix, Suffix of the voucher if required. *

    7. Select the account and debit or credit from the dropdown.

    8. Click Update to save the voucher.

    A confirmation message is displayed.

     

    Other Tasks

    To edit a Voucher Type:

    1. Select the voucher from the tree in the left panel of the voucher master screen.

    2. Click Edit button.

    Now the Add, Edit and Delete button disappears and Update and Cancel button appears.

    3. Make the required changes and click Update.

    Displays the confirmation message.

    To delete a Voucher Type:

    1. Select the voucher from the tree in the left panel of the voucher master screen.

    2. Click Delete button.

    Now the Add, Edit and Delete button is replaced with Confirm Delete and Cancel button.

    3. Click Confirm Delete.

    Displays the confirmation message.

    Note:After deletion the Confirm Delete button becomes inactive.
