    Account Head Master

    This is a master screen. Users can use this for creating Budget Account Heads, sub-heads and mapping them with the Account Heads and Ledgers from the Amrita FIIS. This mapping is used in generating the Budget Vs Actuals’ report. This can be termed as the first process in creating the budgets using Amrita FIIS.

    1. Select Account Head Master sub-menu from the Budget main-menu

    Account Head Master page appears with field Name, combo Account Group and a check box against field sub-account. Two inactive Combo fields Parent and Ledger can be seen. Three buttons Search, Save and Clear buttons can also be seen in the page. Names of the Account Groups created in the FIIS will appear in the Account Group combo.

    2. Enter a name against the field ‘Name’

    3. Select an Account Group from the Account Group Combo

    4. Click ‘Save’

    5. A message ‘New AccountHead added successfully’ appears on the header part of the page

    6. Click ‘Search’, to view the already created Account Heads

    7. Click on the ‘Check’ box against ‘Sub head’.

    The Parent and Ledger combos get activated. The Parent combo lists Account Head created using the budget Account Head Master. Ledger combo lists down the ledger accounts created in the FIIS.

    8. Enter a name against the field ‘Name’

    9. Select the required group from the ‘Parent’ combo

    The Finance account Group mapped to the Budget Account Group gets automatically populated against the Account Group

    10. Select a ledger from the ‘Ledger’ combo

    11. Click ‘Save’

    12. A message ‘New AccountHead added successfully’ appears on the header part of the page

    Other Tasks

    • Searching Account Head/Sub-Head:

    1. Search with or with out an account group being selected

    a. Click ‘Search’

    The ‘Account Head’, ‘Account Group’, Sub Head’, ‘Parent’ and ‘Ledger’ details created by the users under the selected account group is displayed in the data field. There is an ‘Edit’ button at the end of the each data field.

    2. Search using ‘Sub Head’

    i. Check mark the sub head

    ii. Click search

    The ‘Account Head’, ‘Account Group’, Sub Head’, ‘Parent’ and ‘Ledger’ details created by the users under the selected sub head is displayed in the data field. There is an ‘Edit’ button at the end of the each data field.

    a. Users can search using a combination of sub head, Parent and Ledger combination to get the budget heads created under that combination.

    • Editing Account Head/Sub-Head:

    1. Search for account heads/sub heads as explained above.

    The search result appears in the data table. The user can see ‘Search’, ‘Save’ and ‘Clear’ buttons on the page

    2. Click on the ‘Edit’ button at the end of the data table which is to be edited

    The selected value is displayed available for editing. Observe that the ‘Save’ button is replaced by ‘Update’ button.

    3. Make the required changes and click ‘Update’; a message ‘Account Head updated successfully’ appears on top left hand side of the page.
